## Getting started

### 0. Prerequisites

pnpm을 설치했다고 가정합니다.

### 1. Install

```bash
pnpm install
```

### 2. Create SSL cert

로컬 https 개발환경을 위해 인증서를 생성합니다.

```bash
pnpm cert
```

### 3. Start

```bash
pnpm start
```
